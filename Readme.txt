I have included all the code and sql files in the folder.


All the requirements provided by professor has been fulfilled.


In this project, the folloeing things can be done:

1. A user can sign up.
2. A username and password authentication has been done.
3. A user can add friends.
4. A user can delete friends.
5. The chat box will display the list of online users, offline users, friends.
6. A user relogging in  the same browser will open up the same session.
7. A user will recieve the notification for unread messages,
8. Two users can chat with eachother(2-way chat).
9. The user can see the status of online, offline friends.
